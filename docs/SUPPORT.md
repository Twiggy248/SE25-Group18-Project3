# Support

## How to Get Help

If you need help using ReqEngine, you can:

1.  **Check the Documentation:** Review the [README.md](../README.md) and other documentation files for setup guides, usage instructions, and troubleshooting tips.
2.  **Search Existing Issues:** See if your problem has already been reported or solved in our [GitHub Issues](https://github.com/Pradyumna-Chacham/CSC510-SE-Group17/issues).
3.  **Open a New Issue:** If you can't find a solution, please [open a new issue](https://github.com/Pradyumna-Chacham/CSC510-SE-Group17/issues/new) describing your problem in detail.
