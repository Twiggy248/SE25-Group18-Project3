## üåü Project Overview & Key Features 

**ReqEngine** is an intelligent requirements engineering tool built on **FastAPI** that uses a fine-tuned **Large Language Model (LLaMA 3.2 3B Instruct)** to automatically transform unstructured textual requirements into structured, high-quality **Use Case Specifications**. It is engineered for efficiency, accuracy, and reliability across documents of any size.

Key Features include: 
- üß† Intelligent Requirements Processing with smart use case extractions, dynamic token budgeting, duplicate detection, and multi-format document support 
- üìä Advanced Analytics with quality validation, interactive refinement, natural language queries, and sessions management
- üöÄ Export Capabilities like Microsoft Word and Markdown
